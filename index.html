Create New Event

i
Event Name
Choose a catchy name for your event. Examples: "Saturday Night Mixer", "Speed Dating Night", "Singles Mingle". This will be displayed to users in the app.
i
Venue Selection
Start typing to search for a venue. The venue determines where users must be located to join your event.
Search for your venue or select a saved one
i
Event Start Time
When users can start joining your event and drawing cards. Set this 30-60 minutes before you want the main activity to begin.
i
Event End Time
When the event stops accepting new participants and card matching ends. Users can no longer join after this time.
i
Free Drinks Budget
How many free drinks you want to give away. This sets both the variety of cards and maximum matches allowed.
i
Matches Limit
Automatically set to match your drinks budget. This ensures cost control.
i
Event Type
Hetero: Only male-female matches allowed
Gay: Same-gender matches (M/M and F/F)
Random: Anyone can match with anyone regardless of gender. Perfect for networking events.
